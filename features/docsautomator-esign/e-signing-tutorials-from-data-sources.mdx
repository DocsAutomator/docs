Learn on this page how to get documents signed from any of DocsAutomator's data sources. Each tutorial walks you through the full setup process, from configuring your e-signature fields to sending documents for signing.

## Airtable

This tutorial shows how to set up e-signing for documents generated from Airtable. You'll learn how to configure signature placeholders in your template, map signer information from your Airtable records, and trigger the signing workflow automatically when a document is created.
