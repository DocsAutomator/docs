## What is DocsAutomator?

DocsAutomator is a flexible document automation tool that integrates natively with multiple data sources including Airtable, Google Sheets, SmartSuite, Glide, Noloco, ClickUp, and more. The platform also provides an API for custom app integration. Choose between **Google Doc templates** (text-based placeholders) or **PDF templates** (upload an existing PDF and position data fields on top with a visual editor). Generated documents can be saved, emailed, merged, or routed through an **e-signature workflow** for legally binding signatures.

## Document Types Supported

DocsAutomator accommodates nearly any document type. Users automate:

- Invoices and receipts
- Reports and analytics
- Legal documents and contracts
- Marketing materials
- Project plans
- Research papers
- Technical documentation
- Proposals and quotes
