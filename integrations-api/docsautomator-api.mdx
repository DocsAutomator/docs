DocsAutomator offers a powerful REST API to create documents programmatically and manage automations through CRUD operations. The API can be utilized to add a performant document creation solution into any software.

## Base URL

```
https://api.docsautomator.co
```

## Authorization

API authentication uses bearer tokens in the authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

<Info>
Find your API key at [app.docsautomator.co/settings/workspace/api](https://app.docsautomator.co/settings/workspace/api)
</Info>

## Common Error Responses

| Code | Issue |
|------|-------|
| 401 | Incorrect or missing API key |
| 401 | Expired Google refresh token (valid for 60 days) |
| 400 | Missing automation ID |
| 404 | Automation ID not found |
| 404 | No template configured |
| 402 | Document limit exceeded |
| 404 | Automation inactive |

## Endpoints

### Create Document

<ParamField path="POST" type="/createDocument">
  Triggers document generation and returns PDF/Google Doc URLs.
</ParamField>

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `docId` | string | Yes | Automation identifier |
| `documentName` | string | No | Custom document name |
| `data` | object | No | Placeholder values object |
| `line_items_1` | array | No | First line items array |
| `line_items_2` | array | No | Second line items array |

**Line Items with Nesting:**

Pass line items as arrays. Nested items use the reserved `children` property (supports up to 3 levels).

<CodeGroup>
```json Request Body
{
  "docId": "abc123",
  "documentName": "Invoice #001",
  "data": {
    "customer_name": "Acme Corp",
    "invoice_date": "2024-01-15",
    "total": "$1,500.00"
  },
  "line_items_1": [
    {
      "description": "Consulting Services",
      "quantity": 10,
      "rate": 150,
      "amount": 1500
    }
  ]
}
```

```json Response
{
  "pdfUrl": "https://firebasestorage.googleapis.com/...",
  "googleDocUrl": "https://docs.google.com/document/d/...",
  "savePdfGoogleDriveFolderId": "..."
}
```
</CodeGroup>

### Get Automations

<ParamField path="GET" type="/automations">
  Retrieves all workspace automations with configuration details.
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.docsautomator.co/automations" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Response
{
  "automations": [
    {
      "id": "abc123",
      "title": "Invoice Generator",
      "dataSource": "API",
      "status": "active"
    }
  ]
}
```
</CodeGroup>

### List Placeholders

<ParamField path="GET" type="/listPlaceholdersV2">
  Lists available placeholders for main document and line items.
</ParamField>

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `automationId` | string | Yes | The automation ID |

<CodeGroup>
```bash cURL
curl -X GET "https://api.docsautomator.co/listPlaceholdersV2?automationId=abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Delete Automation

<ParamField path="DELETE" type="/deleteAutomation">
  Removes an automation.
</ParamField>

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `docId` | string | Yes | The automation ID |

<Warning>
This action cannot be undone. The automation and all its settings will be permanently deleted.
</Warning>

### Create Automation

<ParamField path="POST" type="/createAutomation">
  Creates new automation with data source.
</ParamField>

**Required Fields:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `title` | string | Yes | Automation name |
| `dataSourceName` | string | Yes | Data source type |
| `docTemplateLink` | string | No | Google Doc URL |

**Valid Data Source Names:**
- `Airtable`
- `API`
- `ClickUp`
- `Glide`
- `Google Sheets`
- `n8n`
- `Noloco`
- `SmartSuite`
- `Zapier`
