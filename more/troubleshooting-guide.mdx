## Quick Reference

| Issue | Data Source | Solution |
|-------|-------------|----------|
| 120 seconds limit | Airtable | Enable async mode |
| Attachment URL disallowed | Airtable | Add wait mechanism |
| No record ID given | Airtable | Configure recId variable |
| Shared Drive permissions | All | Upgrade to Manager role |
| Can't see all bases | Airtable | Reconnect account |
| Webhook formula #Error | Google Sheets | Enable English formulas |
